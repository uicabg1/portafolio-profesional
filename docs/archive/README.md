# Archivo histórico

Esta carpeta conserva documentación útil para trazabilidad, pero que ya no debe tratarse como fuente canónica del estado actual del proyecto.

## Subcarpetas

- `sprints/`
  Runbooks y planes por sprint usados durante la construcción incremental histórica del proyecto.
- `superseded/`
  Documentos reemplazados por versiones nuevas o por documentación canónica más específica.

## Regla de uso

Si un documento de `archive/` contradice a uno en `product/`, `architecture/`, `operations/` o `quality/`, prevalece la documentación canónica.

Si necesitas el sprint activo o el siguiente sprint ejecutable, consulta `docs/sprints/`, no `docs/archive/sprints/`.
